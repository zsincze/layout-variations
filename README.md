# Layout Variations

Layout experiment playground: create and edit grid-based compositions. Compositions are stored in `compositions.json` (created on first run; not committed).

## Run locally

```bash
npm install
npm start
```

Then open **http://localhost:3000** — use **Compositions** to list/create/open, or **Open editor** to work without a composition.

## Ignored by git

- `node_modules/`
- `compositions.json` (local/dev data)
- `.env`
